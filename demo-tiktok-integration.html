TikTok Integration Demo

This page demonstrates the end-to-end flow of the TrailFusion AI web application using TikTok Login Kit and Content Posting API in sandbox mode. The requested scopes are user.info.basic, video.upload, and video.publish. Website domain: trailfusionai.com.

1

Login Kit · Connect TikTok Account

Product: Login Kit

Redirects to TikTok's sandbox OAuth 2.0 authorization endpoint open.tiktokapis.com/v2/oauth/authorize/ with requested scopes.

GET https://www.tiktok.com/v2/auth/authorize/?client_key=SANDBOX_KEY
&scope=user.info.basic,video.upload,video.publish
&response_type=code&redirect_uri=https://trailfusionai.com/callback
TikTok
SANDBOX OAuth 2.0 · Authorize TrailFusion AI

TrailFusion AI is requesting permission to:

  • ✓ Access basic profile info (user.info.basic)
  • ✓ Upload videos to your account (video.upload)
  • ✓ Publish videos on your behalf (video.publish)
2

Scope: user.info.basic

Product: Login Kit

After OAuth, TrailFusion AI calls GET /v2/user/info/ to retrieve the authenticated user's basic profile.

3

Scope: video.upload

Product: Content Posting API

TrailFusion AI calls POST /v2/post/publish/video/init/ and uploads the file to the URL returned by TikTok sandbox.

POST https://open.tiktokapis.com/v2/post/publish/video/init/ Authorization: Bearer <sandbox_access_token> { "post_info": { "title": "TrailFusion AI sandbox demo", "privacy_level": "SELF_ONLY" }, "source_info": { "source": "FILE_UPLOAD", "video_size": 3140928, "chunk_size": 3140928, "total_chunk_count": 1 } }
Response 200 { "data": { "publish_id": "v_pub_url~v2.1776987654321", "upload_url": "https://..." } }

✓ Upload complete · chunk 1/1 uploaded

4

Scope: video.publish

Product: Content Posting API

TrailFusion AI polls POST /v2/post/publish/status/fetch/ until the post completes.
