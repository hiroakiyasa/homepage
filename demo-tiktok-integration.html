TikTok Integration Demo

This page demonstrates the end-to-end flow of the TrailFusion AI application using the TikTok Content Posting API in sandbox mode. The requested scopes are user.info.basic, video.upload, and video.publish.

1

Connect TikTok Account

Click below to start the OAuth flow in sandbox mode. TrailFusion AI will request user.info.basic, video.upload, and video.publish.

2

Scope: user.info.basic

TrailFusion AI retrieves the authenticated user's basic profile.

@trailfusionai_demo

Display name: TrailFusion AI Demo User (sandbox)

open_id: -000GvWPsL59gbtswlOLZLh6...

3

Scope: video.upload

TrailFusion AI calls POST /v2/post/publish/video/init/ and uploads the file to the URL returned by TikTok.

POST https://open.tiktokapis.com/v2/post/publish/video/init/ Authorization: Bearer <sandbox_access_token> { "post_info": { "title": "TrailFusion AI sandbox demo", "privacy_level": "SELF_ONLY" }, "source_info": { "source": "FILE_UPLOAD", "video_size": 3140928, "chunk_size": 3140928, "total_chunk_count": 1 } }
Response 200 { "data": { "publish_id": "v_pub_url~v2.1776987654321", "upload_url": "https://..." } }

4

Scope: video.publish

TrailFusion AI polls POST /v2/post/publish/status/fetch/ until the post completes.
